# Webアプリ
LAMP(ランプ):
Webアプリを開発する上で必要なソフトウェアの組み合わせ。

LAMP:
    L:Kinux(OS)
    A:Apache(Webサーバ)
    M:MySQl(データベース)
    P:PHP(PHP|Perl|Python)

SQLとリレーショナルデータベース:
SQL:
リレーショナルデータっベースを操作するための言語。

リレーショナルデータベース:
表形式でデータを管理するタイプのデータベース。

CRUD(クラッド):
Webアプリに限らずプログラミングでアプリを作成する際の4大機能
- C:Create、登録機能
- R:Read、読み出し機能
- U:Update、変更機能
- D:Delete、削除機能

1.データベースに対して書き込み
2.内容を閲覧
3.内容の変更
4.削除

LAMP環境におけるサーバーサイド処理の流れ:
1. リクエスト
2. プログラムを起動
3. レスポンス
4. HTMLの処理
5. リクエスト
6. プログラムを起動
7. CRUD処理


アルゴリズム:
例:料理
- 使用する材料の情報:データ
- 調理の手順:アルゴリズム
与えられたデータを元に処理を行う。

データ構造:
大量のデータを効率よく管理する仕組みのこと。
